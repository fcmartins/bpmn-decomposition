Dependencies
============

For cvm-lang:

 * tyco-common
 
 
For cvm-app:

 * commons-cli 

Getting tyco-common
===================

CVS URL:

  :extssh:YOURUSERNAME@cvs.dcc.fc.up.pt:/home/cvsroot/tyco/common

Read `common/README-tycocommon.txt' for information on usage.

Setup the TycoCommon path:

  echo tycocommon.path = PATH/TO/common > build.properties

Getting commons-cli
===================

The Apache Commons CLI library provides an API for parsing command line options
passed to programs.

URL: http://commons.apache.org/cli/

Place the JAR in 'lib' directory with name 'commons-cli.jar'.

Usage
=====

Run `ant -p` for more info.

Generated files show up in directory 'target.'
